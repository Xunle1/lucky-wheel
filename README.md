# 命运酒单转盘 (Pixel Bar Edition)

一个像素风格的在线老虎机，帮你决定今晚喝什么！

## 特性
- 🎰 **像素艺术风格**：复古 8-bit 设计。
- 🕹️ **交互式拉杆**：点击拉杆启动转盘。
- 🎵 **音效**：自带 8-bit 芯片音乐音效。
- 📝 **可配置**：修改 `config.js` 即可自定义酒单。
- 📱 **响应式**：手机和电脑都能用。

## 如何修改酒单
1. 打开 `config.js` 文件。
2. 修改 `DRINKS` 列表中的文字。
3. 保存即可。

## 部署
本项目已针对 GitHub Pages 优化。
只需推送到 `main` 分支即可自动部署。

## 本地运行
直接双击打开 `index.html` 即可（部分浏览器可能因安全策略限制音频，建议使用本地服务器）：
```bash
python3 -m http.server
```
然后访问 `http://localhost:8000`。
